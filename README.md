# Curso de OTGA 2026
## Visualizations with Python (curso en español)

### Materiales, ejemplos y conjuntos de datos utilizados en el curso de Visualización de Datos en Python.

#### Objetivos

Este repositorio reúne los recursos necesarios para aprender un flujo de trabajo completo de análisis y visualización de datos utilizando Python, desde la obtención de datos hasta la construcción de visualizaciones estáticas, interactivas y geoespaciales.

Durante el curso se abordarán temas como:

+ Obtención de datos desde repositorios públicos.
+ Uso de Google Colab como entorno de trabajo.
+ Manipulación y limpieza de datos con Pandas y Polars.
+ Análisis exploratorio de datos (EDA).
+ Vis1ualización con Matplotlib y Seaborn.
+ Visualización declarativa con Altair.
+ Visualización interactiva con Plotly.
+ Cartografía interactiva con Folium y Leaflet.

# Curso de OTGA 2026
## Visualizations with Python (curso en español)

### Materiales, ejemplos y conjuntos de datos utilizados en el curso de Visualización de Datos en Python.

## Objetivos

Este repositorio reúne los recursos necesarios para desarrollar un flujo de trabajo completo de análisis y visualización de datos utilizando Python.

El objetivo es recorrer todas las etapas del proceso: desde la obtención y preparación de datos, pasando por el análisis exploratorio, hasta la creación de visualizaciones estáticas, interactivas y geoespaciales.

Durante el curso se trabajará con datos reales y herramientas ampliamente utilizadas en ciencia de datos.

## Contenidos

### 1. Introducción al ecosistema Python para datos

- Google Colab como entorno de desarrollo
- Estructura de notebooks reproducibles
- Instalación y gestión de librerías
- Importación de datos desde archivos y repositorios públicos

Librerías:
- NumPy
- Pandas
- Polars


### 2. Preparación y exploración de datos

- Lectura de datos CSV, Excel y formatos geoespaciales
- Inspección de estructuras de datos
- Limpieza y transformación
- Manejo de valores faltantes
- Selección y filtrado de información

Conceptos:
- DataFrames
- Tipos de variables
- Estadística descriptiva
- Análisis exploratorio de datos (EDA)


### 3. Visualización estática

Creación de gráficos utilizando:

- Matplotlib
- Seaborn

Se abordarán:

- Gráficos de líneas
- Barras
- Dispersión
- Histogramas
- Distribuciones
- Matrices de correlación

Buenas prácticas:

- Elección del gráfico adecuado
- Uso del color
- Etiquetas y escalas
- Errores frecuentes en visualización


### 4. Visualización declarativa e interactiva

Visualización basada en gramáticas gráficas:

- Altair

Visualización interactiva:

- Plotly

Conceptos:

- Gráficos dinámicos
- Tooltips
- Selecciones
- Filtros
- Exportación a HTML


### 5. Visualización geoespacial

Representación de datos espaciales:

- Sistemas de coordenadas
- Datos vectoriales
- Datos puntuales y polígonos

Herramientas:

- GeoPandas
- Folium
- Leaflet

Aplicaciones:

- Mapas interactivos
- Distribución espacial de variables
- Análisis territorial


### 6. Proyecto final

Desarrollo de un flujo completo:

1. Selección u obtención de datos
2. Limpieza y preparación
3. Análisis exploratorio
4. Diseño de visualizaciones
5. Construcción de una narrativa visual

El proyecto deberá incluir al menos:

- Una visualización estática
- Una visualización interactiva
- Una visualización geoespacial (cuando aplique)
